1. Посмотреть где я - команда pwd 
2. Создать папку -  mkdir homework_1
3.Зайти в папку -  cd homework_1
4. Создать 3 папки - командой mkdir lesson_1 lesson_2 lesson_3
5. Зайти в любую папку: cd lesson_1
6. Создайте 5 файлов: команда touch. touch tt.txt tt_2.txt  tt_3.txt xx.json xx_2.json
7. Создать 3 папки - mkdir papkav_1 papkav_2 papkav_3
8. Вывести список содержимого из папки - команда ls -la (нужно зайти в саму папку ( cd papkav_1, и применить команду ls -la). 
9. Открыть любой txt файл - открываем командой vim и название файла, vim tt.txt
10. Написать туда что нибудь, любой текст. Входим в режим insert (нужно нажать i) и теперь можно что то писать.
11. Сохранить и выйти - Выходим из режима insert и жмем esc. Потом пишем команду - :wq.
12. Выйти из папки на уровень выше - команда cd ..
13. Переместить любые 2 файла, которые вы создали, в любую другую папку - команда mv. mv tt.txt tt_2.txt papkav_2/ 
14. Скопировать любые 2 файла, которые вы создали, в любую другую папку - команда cp. cp tt_3.txt xx.json papkav_3/
15. Найти файл по имени - команда find. find -name tt.txt
16. Посмотреть содержимое в реальном времени - команда grep. tail -f tt.txt|grep""
17. Вывести несколько первых строк из текстового редактора - команда head. head -2 tt.txt
18. Вывести несколько последних строк из текстового редактора - команда tail. tail -2 tt.txt
19. Посмотреть содержимое длинного файла - команда less. less tt.txt
20. Вывести дату и время - команда date
